Dataset Description

Question 2: Popularity score vs runtime - Does the runtime affect the popularity of a movie?

Question 3: Average profitability by genre - What genres are generally more profitable?

# Function to make scatter plots
def sc_plot(df, x_column, y_column, title, x_label, y_label):
    df.plot(kind = 'scatter', x = x_column, y = y_column)
    plt. title(title)
    plt.xlabel(x_label)
    plt.ylabel(y_label)
    plt.show()

Creating function to call for the next two scatter plots.

In[29]:
# Using the df_genres dataframe to show average profit by genre
avg_profit_by_genre = df_genres.groupby('genres')['profit'].mean().sort_values(ascending=True)

avg_profit_by_genre = avg_profit_by_genre / 10000000

Animation, adventure, and family genre movies are the most profitable genres. Other genres can be successful, but possibly due to other things such as budget have overall lower average profit.

Conclusions

A Good popularity score could indicate a chance for increased profitability. Too long of a runtime appears to negatively affect the popularity score. Animation, adventure, and family genres have the top three average profitabilities. All of this is with a dataset that was cleaned and possibly skewed by removing nulls and 0s. To make correlative analysis, I would request more complete data.
